# Captive Portal Improv

CLI for automating the process of connecting an Esphome device to a WiFi network via Captive Portal.

```bash
bun install
```

To run:

```bash
bun start
```

This project was created using `bun init` in bun v1.3.9. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
